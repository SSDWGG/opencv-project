# 实时发型试戴

这是一个浏览器端实时摄像头发型试戴原型：通过人脸关键点检测定位额头、太阳穴、脸宽和头部倾斜角，再用 Canvas 合成真实透明 PNG 发型覆盖层。当前内置「空气感短卷」「侧扫层次刘海」「顺直齐刘海」「长卷披肩」四种目标发型，并支持开启拟人动物隐私贴图来遮挡真实面部内容。

## 运行

```bash
npm install
npm run dev
```

打开终端输出的本地地址后，点击「开启摄像头」并允许浏览器访问摄像头。

## 技术方案

- `@mediapipe/tasks-vision`：实时检测 478 个人脸关键点，比纯 OpenCV Haar 检测更适合发型贴合和头部姿态估计。
- `Canvas 2D`：绘制镜像摄像头画面，并按脸宽、额头位置和眼角连线角度实时缩放/旋转真实发丝 PNG。
- 平滑跟踪：对位置、尺寸和旋转角做插值，减少摄像头抖动造成的发型闪烁。
- 真实感合成：发型图层会先绘制柔化暗部遮罩，再叠加透明发丝素材、投影和额头接触阴影；发色选择器只做轻微混合，尽量保留原素材的高光和发丝纹理。
- 隐私贴图：默认开启小狐脸遮挡，贴图会跟随头部旋转、眨眼和张嘴变化；需要查看原始面部时可在侧边栏关闭。

## 发型素材

发型图片放在 `public/hair/`，来源为 PurePNG 的透明 PNG 素材，并已裁掉多余透明边界以便实时贴合。详情见 `public/hair/README.md`。

## 当前限制

- 当前版本是实时试戴合成，不会真正擦除原始头发。
- 隐私贴图是本地 Canvas 覆盖层，不会上传摄像头画面，但关闭后画布仍会显示原始面部。
- 首次加载需要联网下载 MediaPipe wasm 和人脸关键点模型。
- 如果要做到更接近“换发”效果，下一步可以增加头发分割模型或 OpenCV.js 的局部遮罩/GrabCut 后处理。
